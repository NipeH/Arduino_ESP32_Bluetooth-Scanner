## Getting Started

Näillä ohjeilla saat projektisi pyörimään.

1. ESP32 pitää uudelleen formatoida jotta ohjelma mahtuu laitteelle.
   
   Tässä ohje: https://www.youtube.com/watch?v=5VoXNloOwZE

2. Install NPM packages
   ```sh
   npm install
   ```
